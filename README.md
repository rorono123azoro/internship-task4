# Internship Task 4: Basic REST API

A functional REST API built with Node.js and Express to learn backend fundamentals.

## Features
- Complete CRUD operations (Create, Read, Update, Delete)
- API endpoint for a `User` entity (name, email, age)
- Ready for interaction via Postman or other REST clients

## How to run
1. Ensure Node.js is installed.
2. Run `npm install` to install dependencies.
3. Run `node server.js` (or your entry file) to start the API server.
